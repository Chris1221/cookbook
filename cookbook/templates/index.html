Technical summary

The application is fully written in flask, though I wanted to host it on a static server. To accomplish this, I used a combination of Flask-Freeze and CirlceCI with gh-pages to both make the site static and automatically deploy to github pages whenever I push to the repository. Recipes are described through an extremely simple YAML markup with flexible fields that can be omitted if desired. Tags are rendered out seperately and are linked to the recipes they belong to. Other than that, I lifted some CSS from the al-folio jekyll theme that I use for my main site. If curious, the website is open source on Github. {% endblock %}
